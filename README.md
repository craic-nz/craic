## What this is

A small, practical collection of reusable resources: AI prompts, blank templates and plain-English guides for everyday information work in New Zealand Government. Everything is generic, free to reuse, and written so it can be adapted to any agency without naming one.

There is nothing to buy and no account to create. The website is a static front end over this repository; the repository is the source of truth.

## Repository structure

```
craic/
  index.html        Home
  library.html      Searchable catalogue of all resources
  resource.html     In-browser viewer for any library item
  about.html        About, contributions and disclaimer
  assets/           Brand, styles and scripts
  library/
    prompts/        AI prompts
    templates/      Blank, fillable templates
    guides/         Quick-reference guides and curated links
```

## Contributing

Corrections and new resources are welcome by email, issue or pull request. Please keep contributions generic, reusable, and free of any agency-specific, commercial or personal detail.
